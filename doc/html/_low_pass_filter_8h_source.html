00027 /*
00028 simple resonant filter posted to musicdsp.org by Paul Kellett http://www.musicdsp.org/archive.php?classid=3#259
00029 
00030 // set feedback amount given f and q between 0 and 1
00031 fb = q + q/(1.0 - f);
00032 
00033 // for each sample...
00034 buf0 = buf0 + f * (in - buf0 + fb * (buf0 - buf1));
00035 buf1 = buf1 + f * (buf0 - buf1);
00036 out = buf1;
00037 
00038 fixed point version of the filter
00039 "dave's blog of art and programming" http://www.pawfal.org/dave/blog/2011/09/
00040 */
